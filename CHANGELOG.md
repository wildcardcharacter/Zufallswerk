# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-06-26

### Added

- First public release of Zufallswerk
- Secure password generation using `/dev/urandom`
- Lightweight graphical user interface using YAD
- Configurable password length
- Selectable character sets
  - Lowercase letters
  - Uppercase letters
  - Numbers
  - Special characters
- Automatic clipboard integration via `xclip`
- Password strength indicator
- Support for generating multiple passwords without restarting the application
- XFCE application menu integration
- Custom application icon
- Debian package build script
- GitHub release with `.deb` package
